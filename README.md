# gff3-validator-flask
Flask/Docker Implementation of GFF3 Validator

## 

Run Docker Container
```
docker run --publish 5000:5000 gff3-validator
```

Run in detached mode
```
docker run --publish 5000:5000 gff3-validator
```

Dev Compose Locally
```
docker-compose -f docker-compose.dev.yml up --build
```
